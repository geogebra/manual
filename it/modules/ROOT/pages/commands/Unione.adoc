Unione(Lista, Lista)::
  Unisce le due liste e rimuove gli elementi ripetuti.

[EXAMPLE]
====

`++Unione({1, 2, 3, 4, 5}, {3, 2, 1, 7})++` restituisce `++{1, 2, 3, 4, 5, 7}++`

====
