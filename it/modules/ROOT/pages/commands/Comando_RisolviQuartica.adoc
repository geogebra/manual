== [#Sintassi_CAS]#Sintassi CAS#

RisolviQuartica(Polinomio quartico)::
  Restituisce la lista delle radici reali e complesse del polinomio di 4° grado indicato.

[EXAMPLE]
====

`++RisolviQuartica(x^4+x^3+x^2+x)++` restituisce \{0, -1, _i_, -_i_}.

====
