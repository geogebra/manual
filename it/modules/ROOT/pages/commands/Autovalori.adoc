= Comando Autovalori

== Sintassi CAS

Autovalori( Matrice )::
  Determina gli autovalori della matrice indicata.

[EXAMPLE]
====

`++Autovalori({{1, 2}, {3, 4}})++` restituisce stem:[ \left\\{ \frac\{\sqrt\{33} + 5}\{2}, \frac\{-\sqrt\{33} + 5}\{2}
\right\} ]

====
