Raggio(Circonferenza)::
  Calcola il raggio della circonferenza.

[EXAMPLE]
====

*Esempi:*

* Sia _c_ la circonferenza di equazione (x - 1)² + (y - 1)² = 9. Allora `++Raggio(c)++` restituisce _a_ = 3
* `++Raggio((x-2)²+(y-2)²=16)++` restituisce _a_ = 4.
