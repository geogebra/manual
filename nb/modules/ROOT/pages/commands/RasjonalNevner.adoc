RasjonalNevner[ <Tall> ]::
  Gjør om tallet til en brøk med rasjonal nevner.

[EXAMPLE]
====

* `++RasjonalNevner[3.5]++` gir stem:[\frac\{7}\{2}].
* `++RasjonalNevner[1/sqrt(2)]++` gir stem:[\frac\{\sqrt\{2} }\{2}].

====

[NOTE]
====

*Merk:* Se også kommandoen xref:/commands/BlandetTall.adoc[BlandetTall].
