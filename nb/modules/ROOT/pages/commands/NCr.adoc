== CAS-delen

nCr[ <Tall n>, <Tall r> ]

Beregner binomialkoeffisienten stem:[ n \choose r ].

[EXAMPLE]
====

`++nCr[5,3]++` gir verdien
stem:[\frac\{5!}\{3!(5-3)!}=\frac\{5!}\{3!2!}=\frac\{5*4*3*2*1}\{3*2*1*2*1}=\frac\{5*4}\{2}=5*2=10].

====
