== CAS Syntax

BlandetTall[ <Tall> ]::
  Konverterer det gitte tallet til et blandet tall.

[EXAMPLE]
====

* `++BlandetTall[3.5]++` returnerer stem:[3 + \frac\{1}\{2}].
* `++BlandetTall[12 / 3]++` returnerer _4_.
* `++BlandetTall[12 / 14]++` returnerer stem:[\frac\{6}\{7}].
