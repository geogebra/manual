= Reverse Command

Reverse( <List> )::
  Reverses the order of a xref:/Lists.adoc[list].

[EXAMPLE]
====

`++Reverse(list1)++` reverses `++list1 = {(1, 2), (3, 4), (5, 6)}++` to create _list2 = \{(5, 6), (3, 4), (1, 2)}_

====

== CAS Syntax

Reverse( <List> )::
  Reverses the order of a xref:/Lists.adoc[list].

[EXAMPLE]
====

`++Reverse({1, 2, 3, 4})++` reverses the list to create _\{4, 3, 2, 1}_

====
