= Extrude to Prism or Cylinder Tool
:page-en: tools/Extrude_to_Prism_or_Cylinder
ifdef::env-github[:imagesdir: /en/modules/ROOT/assets/images]

Activate the tool, then:

* Select and drag a polygon or a circle to create a right prism or cylinder.

or

* Select a polygon or a circle, then enter the value of the _altitude_ in the appearing input box to create a right prism or cylinder. The checkbox _Positive values follow the view_ is active by default and sets the positive direction of the altitude of the prism or cylinder towards the user.
