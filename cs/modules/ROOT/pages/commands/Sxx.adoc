:page-en: commands/Sxx

SXX( <Seznam čísel> )

Vypočítá statistiku stem:[\sum x^2 - \frac{(\sum x)^2}{n}].

SXX( <Seznam bodů> )

Vypočítá statistiku stem:[\sum x^2 - \frac{(\sum x)^2}{n}] podle _x_-ových souřadnic zadaných bodů.
