== CAS Syntax

VlastniVektory( <Matice> )::
  Najde vlastní vektory dané matice.

[EXAMPLE]
====

`++VlastniVektory({{1, 2}, {3, 4}})++` dává stem:[ \left(\begin{array}\sqrt{33} - 3&-\sqrt{33} -
3\\6&6\\\end{array}\right) ]

====
