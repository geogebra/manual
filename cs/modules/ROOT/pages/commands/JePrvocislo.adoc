JePrvocislo( <Číslo> )::
  Výsledkem je _true_ nebo _false_ podle toho, zda je zadané _Číslo_
  https://cs.wikipedia.org/wiki/Prvo%C4%8D%C3%ADslo[prvočíslem].

[EXAMPLE]
====

* `++JePrvočíslo[10]++` dává _false_.
* `++JePrvočíslo[11]++` dává _true_.

====
