== CAS Syntax

VlastniCisla( <Matice> )::
  Najde vlastní čísla dané matice.

[EXAMPLE]
====

`++VlastniCisla({{1, 2}, {3, 4}})++` dává stem:[ \left\{ \frac{\sqrt{33} + 5}{2}, \frac{-\sqrt{33} + 5}{2}
\right\} ]
