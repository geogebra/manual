:page-en: commands/LetterToUnicode

ZnakNaUnicode [ "<Znak>" ]::
  Výstupem příkazu je číslo daného znaku v https://cs.wikipedia.org/wiki/Unicode[tabulce Unicode].

[EXAMPLE]
====

`++ZnakNaUnicode["a"]++` dává jako výsledek číslo 97.

====

[NOTE]
====

Znak musí být zadán mezi uvozovkami.

====
