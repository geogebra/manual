== CAS-Ansicht

Eigenwerte( <Matrix> )::
  Berechnet die Eigenwerte der gegebenen Matrix.

[EXAMPLE]
====

`++Eigenwerte({{1, 2}, {3, 4}})++` liefert stem:[ \left\{ \frac{\sqrt{33} + 5}{2}, \frac{-\sqrt{33} + 5}{2}
\right\} ]

====

====

xref:/commands/Transponiere.adoc[Transponiere].
