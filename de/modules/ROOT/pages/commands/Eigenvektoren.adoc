== CAS-Ansicht

Eigenvektoren( <Matrix> )::
  Berechnet die Eigenvektoren der gegebenen Matrix.

[EXAMPLE]
====

`++Eigenvektoren({{1, 2}, {3, 4}})++` liefert stem:[ \left(\begin{array}{}\sqrt{33} - 3&-\sqrt{33} -
3\\6&6\\\end{array}\right) ]

====

[NOTE]
====

* Siehe auch die Befehle xref:/commands/Eigenwerte.adoc[Eigenwerte], xref:/commands/Invertiere.adoc[Invertiere],
xref:/commands/Transponiere.adoc[Transponiere].
