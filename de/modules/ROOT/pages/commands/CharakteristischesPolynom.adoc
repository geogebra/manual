== CAS-Ansicht

CharakteristischesPolynom( <Matrix> )::
  Liefert das https://de.wikipedia.org/wiki/Charakteristisches_Polynom[charakteristische Polynom] der gegebenen Matrix.

[EXAMPLE]
====

`++CharakteristischesPolynom({{1,2},{3,4}})++` liefert stem:[x^2-5x-2].

====
