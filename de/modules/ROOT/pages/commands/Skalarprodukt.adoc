== CAS Syntax

Skalarprodukt( <Vektor oder Liste>, <Vektor oder Liste> )::
  Berechnet das Skalarprodukt zweier Vektoren oder Listen.

[EXAMPLE]
====

Die Eingaben `++Skalarprodukt[(1, 3, 2),(0, 3, -2)]++` un
`++Skalarprodukt[(1, 3, 2), (0, 3, -2)]++` ergeben _5_.

Die Eingabe `++Skalarprodukt[(1, 2), (2, 3)]++` ergibt _8_.

====

[NOTE]
====

Siehe auch Befehl xref:/commands/Kreuzprodukt.adoc[Kreuzprodukt].

====
