= CASLoaded (Befehl)
:page-en: commands/CASLoaded
ifdef::env-github[:imagesdir: /de/modules/ROOT/assets/images]

CASLoaded() ::
  Gibt einen booleschen Wert zurück: `true`, wenn CAS-Befehle bereits geladen wurden, andernfalls `false`.
  Der Wert ist dynamisch (ändert sich auf `true`, sobald die CAS-Befehle geladen sind).

[NOTE]
====
Dieser Befehl ist in der Webversion von GeoGebra nützlich, wo CAS-Befehle mit einer Verzögerung nach dem Start der App
geladen werden. Sie können diesen Befehl zusammen mit
xref:/Bedingte_Sichtbarkeit.adoc[Bedingter Sichtbarkeit] verwenden, um Konstruktionselemente auszublenden, während
das CAS geladen wird.
====
