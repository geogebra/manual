== CAS Syntax

Shuffle( <List> )::
  Returns list with same elements, but in random order.

[EXAMPLE]
====

* `++Shuffle({3, 5, 1, 7, 3})++` yields for example _\{5, 1, 3, 3, 7}_.
* `++Shuffle(Sequence(20))++` gives the first 20 whole numbers in a random order.
