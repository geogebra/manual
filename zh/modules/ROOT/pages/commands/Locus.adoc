Locus[Point Q, Point P]::
  Returns the locus line of point _Q_ which depends on point _P_.

[NOTE]
====
Point _P_ has to be a point on an object (e. g. line, segment, circle).

====

Loci are specific object type which appears in xref:/自變、應變、輔助物件.adoc[auxiliary objects].
Besides Locus command they are result of some
xref:/Discrete_Math_Commands.adoc[Discrete Math Commands] and
xref:/SolveODE_Command.adoc[SolveODE Command]. Loci are
xref:/幾何物件.adoc[paths] and can be used within path-related commands such as
xref:/commands/Point.adoc[Point]. Their properties depend on how they were obtained, see e.g.
xref:/commands/Perimeter.adoc[Perimeter Command] and
xref:/First_Command.adoc[First Command].
