ClosestPoint[Path P, Point A]::
  Returns a point on xref:/幾何物件.adoc[path] _P_ which is the closest to point _A_.

[NOTE]
====
For xref:/函數.adoc[Functions], this command will return the point vertically in line rather than the
nearest point
