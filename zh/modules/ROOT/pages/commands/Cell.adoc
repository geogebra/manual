:page-en: commands/Cell

Cell[ <Column>, <Row> ]::
Cell[ <Column>, <Row> ]::
  Returns copy of xref:/試算表.adoc[spreadsheet] cell in given column and row.

[EXAMPLE]
====
 `++Cell[2,1]++` returns copy of B1.

====

[NOTE]
====
By default the cells in spreadsheet cells are xref:/自變、應變、輔助物件.adoc[auxiliary] and in
such case this command returns auxiliary object as well.

====
