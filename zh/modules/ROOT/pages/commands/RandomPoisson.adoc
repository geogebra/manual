== CAS 視窗

RandomPoisson[ <Mean> ]::
RandomPoisson[ <Mean> ]::
  Generates a random number from a Poisson distribution with given mean.

[EXAMPLE]
====


`++RandomPoisson[3]++` yields a random value from a Poisson distribution with a mean of _3_.

====

[NOTE]
====

*備註：*

See also xref:/RandomBinomial_Command.adoc[RandomBinomial Command] and
xref:/RandomNormal_Command.adoc[RandomNormal Command].

====
