= Vector 指令
:page-en: commands/Vector
ifdef::env-github[:imagesdir: /zh/modules/ROOT/assets/images]

Vector( <點> )::
  回傳指向 <點> 的位置向量。
Vector( <始點>, <終點> )::
  建立一個從 <始點> 到 <終點> 的向量。

[EXAMPLE]
====


`++Vector((1, 1), (3, 4))++` 產生 _u = stem:[\begin\{pmatrix}2\\3\end\{pmatrix}]_。

====

[EXAMPLE]
====


`++Vector((3, 2))++` 產生 _u = stem:[\begin\{pmatrix}3\\2\end\{pmatrix}]_。

====
