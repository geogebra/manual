FractionalPart[ <Expression> ]::
FractionalPart[ <Expression> ]::
  Returns the fractional part of the expression.

[EXAMPLE]
====


* `++FractionalPart[6/5]++` yields stem:[\frac\{1}\{5}],
* `++FractionalPart[1/5 + 3/2 + 2]++` yields stem:[\frac\{7}\{10}].

====
