== Widoczność

Możesz pokazywać lub ukrywać obiekty w image:16px-Menu_view_graphics.svg.png[Menu view graphics.svg,width=16,height=16]
xref:/Widok_Grafiki.adoc[Widoku Grafiki] na różne sposoby.

* Możesz użyć narzędzia image:20px-Mode_showhideobject.svg.png[Mode showhideobject.svg,width=20,height=20]
xref:/tools/Pokaż_Ukryj_obiekt.adoc[Pokaż / Ukryj obiekt], aby wyświetlać i ukrywać obiekty.
* Otwórz xref:/Menu_Kontekstowe.adoc[menu kontekstowe] i wybierz opcję image:20px-Mode_showhideobject.svg.png[Mode
showhideobject.svg,width=20,height=20] _Pokaż obiekt_, aby zmienić widoczność wybranego obiektu.
* W image:16px-Menu_view_algebra.svg.png[Menu view algebra.svg,width=16,height=16] xref:/Widok_Algebra.adoc[Widoku Algebry],
kółko po lewej stronie każdego obiektu pokazuje jego aktualny stan widoczności (widoczny lub ukryty). 
Możesz je kliknąć, aby zmienić stan widoczności obiektu.
* Możesz także użyć narzędzia image:20px-Mode_showcheckbox.svg.png[Mode showcheckbox.svg,width=20,height=20]
xref:/tools/Pole_Wyboru.adoc[Pole wyboru], aby wyświetlać lub ukrywać jeden lub kilka obiektów.

[NOTE]

== Osadzanie obiektów

Obiekt możesz osadzić za pomocą image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16]
_okna Ustawienia obiektu_. Obiektów osadzonych (zarówno xref:/Obiekty_Swobodne_Zależne_i_Pomocnicze.adoc[swobodnych jak i zaleznych]) nir można
przesuwać. Aby uniemożliwić zaznaczanie  lub usuwanie obiektu, możesz odznaczyć opcję "Wybór dozwolony" w zakładce Zaawansowane.

== Wypełnienie

Dla xref:/Krzywe.adoc[krzywych zamkniętych] i xref:/Obiekty_Geometryczne.adoc[obszarów] możesz określić wypełnienie, korzystając z
zakładki _Styl_ w image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] _oknie Ustawienia obiektu_. Dostępne są następujące typy wypełnienia:

Standard::
  Wypełnia obiekt kolorem określonym na karcie Kolor. Ten sam kolor jest używany do rysowania obramowania obiektu. W zakładce _Styl_ możesz określić przezroczystość — 
np. krzywe stożkowe mają domyślnie przezroczystość 0, co oznacza, że są przezroczyste.
Kreska::
  Obiekt jest kreskowany; można określić kąt kresek oraz odległość między nimi. Grubość kresek jest równa grubości obramowania obiektu.
Kratka::
  Obiekt jest wypełniany kratką (dostępny kąt kresek: 0°–45°); można określić odległość między liniami. Grubość kresek jest równa grubości obramowania obiektu.
Szachownica::
  Wypełnia obiekt wzorem szachownicy (dostępny kąt kwadratów: 0°–45°). Można określić wysokość kwadratów.
Kropka::
  Wypełnia obiekt kropkami; można określić odległość między kropkami.
Plaster_miodu::
  Wypełnia obiekt wzorem plastra miodu; można określić odległość między komórkami.
Cegiełki::
  Wypełnia obiekt wzorem cegieł; można określić kąty oraz wysokość cegieł.
Tkanina::
  Wypełnia obiekt wzorem splotu przypominającego tkaninę; można określić kąty oraz odstępy.
Symbol::
  Wypełnia obiekt wybranym z listy symbolem; można określić odległość między symbolami.
Obraz::
  Jako wypełnienie możesz użyć plik graficzny zapisany na twoim lokalnym dysku. Obraz jest powtarzany okresowo, jego rozmiar jest stały (w pikselach), 
a lewy górny róg obrazu jest wyrównany do lewego górnego rogu widoku.

[NOTE]
====

Możesz także zaznaczyć opcję _Odwróć wypełnienie_, aby wybranym wzorem wypełnić cały
image:16px-Menu_view_graphics.svg.png[Menu view graphics.svg,width=16,height=16] _Widok Grafiki_, poza wybranym obiektem.

====
