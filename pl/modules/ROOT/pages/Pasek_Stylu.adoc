== [#Czym_jest_Pasek_stylu?]#Czym jest Pasek stylu?#

_Pasek stylu_ umożliwia szybką i wygodną zmianę wybranych podstawowych właściwości xref:/Podstawowe_widoki.adoc[_widoków_] lub
xref:/Obiekty.adoc[obiektów]. Aby otworzyć lub zamknąć _Pasek stylu_, kliknij małą strzałkę obok nazwy odpowiedniego
xref:/Podstawowe_widoki.adoc[_widoku_] w GeoGebrze Desktop lub kliknij _przycisk Paska stylu_ w aplikacjach GeoGebra Web 
i aplikacjach moblilnych. Zwróć uwagę, że więcej właściwości można zmienić w oknie dialogowym
image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] xref:/Okno_Ustawień_Obiektu.adoc[_Ustawienia_].

== [#Opcje_Paska_stylu_w_różnych_widokach]#Opcje Paska stylu w różnych widokach_#

Każdy xref:/Podstawowe_widoki.adoc[_widok_] ma własne opcje _paska stylu_. W
xref:/GeoGebra_5_0_porównanie_wersji_desktopowej_webowej_i_na_tablet.adoc[GeoGebrze Web i aplikacjach moblilnych], w zależności od wybranego _widoku_,
_pasek stylu_ można otworzyć za pomocą jednego z _przycisków paska stylu_; wyświetlane są wtedy najczęściej używane opcje danego _widoku_.

grid.svg,width=16,height=16] *Pokaż / Ukryj siatkę*: Możesz wyświetlić lub ukryć siatkę (GeoGebra Desktop) albo wybrać typ siatki (GeoGebra Web i aplikacje mobilne).

*Przywróć widok domyślny*: Jeśli image:16px-Menu_view_graphics.svg.png[Menu view
graphics.svg,width=16,height=16] _Widok Grafiki_ został wcześniej zmieniony, ten przycisk przywraca domyślne położenie początku układu współrzędnych.
* image:16px-Stylingbar_graphicsview_point_capturing.svg.png[Stylingbar graphicsview point
capturing.svg,width=16,height=16] *Zmień przyciąganie punktów*: Możesz wybrać jedno z ustawień:

xref:/Przyciąganie_Punktów.adoc[Przyciąganie punktów].
* image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] *Otwórz okno Ustawienia*: Otwiera
image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] _xref:/Okno_Ustawień_Obiektu.adoc[okno Ustawienia]_
dla image:16px-Menu_view_graphics.svg.png[Menu view graphics.svg,width=16,height=16] _Widoku Grafiki_ lub zaznaczonego
obiektu (GeoGebra Web i aplikacje mobilne).

=== image:24px-Stylingbar_icon_algebra.svg.png[Stylingbar icon algebra.svg,width=24,height=17] Pasek stylu Widoku Algebry

Użyj image:16px-Stylingbar_icon_algebra.svg.png[Stylingbar icon algebra.svg,width=16,height=12] _przycisku Paska stylu 
Widoku Algebry_, aby otworzyć _Pasek stylu_ tego widoku oferujący następujące opcje:

* image:16px-Stylingbar_algebraview_auxiliary_objects.svg.png[Stylingbar algebraview auxiliary
objects.svg,width=16,height=16] *Pokaż / Ukryj obiekty pomocnicze*: Pokazuje lub ukrywa 
xref:/Obiekty_Swobodne_Zależne_i_Pomocnicze.adoc[_obiekty pomocnicze_] w image:16px-Menu_view_algebra.svg.png[Menu
view algebra.svg,width=16,height=16] _Widoku Algebry_.
* image:16px-Stylingbar_algebraview_sort_objects_by.svg.png[Stylingbar algebraview sort objects
by.svg,width=16,height=16] *Sortuj obiekty według:*: Możesz uporządkować obiekty matematyczne według następujących kryteriów:
** _Zależność_:Obiekty są podzielone na obiekty xref:/Obiekty_Swobodne_Zależne_i_Pomocnicze.adoc[_swobodne_] 
i xref:/Obiekty_Swobodne_Zależne_i_Pomocnicze.adoc[_zależne_]. Jeśli utworzysz nowy obiekt bez użycia innych istniejących obiektów, 
jest on klasyfikowany jako _obiekt swobodny_.
Jeśli nowo utworzony obiekt powstał z wykorzystaniem innych istniejących obiektów, jest on klasyfikowany jako _obiekt zależny_.
** _Typ obiektu_: Obiekty są uporządkowane według _typu_ (np. kąty, proste, punkty) w kolejności alfabetycznej.
** _Warstwa_: Obiekty są uporządkowane według _xref:/Warstwy.adoc[warstwy]_, na której są rysowane.
** _Kolejność konstrukcji_: Obiekty są uporządkowane według kolejności tworzenia.
* image:16px-Stylingbar_dots.svg.png[Stylingbar dots.svg,width=16,height=16] *Pokaż widoki*: Możesz wyświetlić dodatkowe
xref:/Podstawowe_widoki.adoc[_widoki_] w oknie GeoGebry (GeoGebra Web i aplikacje mobilne).

== [#Opcje_paska_stylu_dla_narzędzi_i_obiektów]#Opcje paska stylu dla narzędzi i obiektów#

W zależności od wybranego xref:/Narzedzia.adoc[_narzędzie_] lub obiektu _pasek stylu_ umożliwia zmianę następujących właściwości
wybranego obiektu/obiektów lub obiektu/obiektów tworzonych za pomocą wybranego
selected xref:/Narzedzia.adoc[_narzędzia_]:

* image:16px-Stylingbar_point.svg.png[Stylingbar point.svg,width=16,height=16] *Styl punktu*: Wybór stylu punktu (np.
image:16px-Stylingbar_point.svg.png[Stylingbar point.svg,width=16,height=16] punkt,
image:16px-Stylingbar_point_cross.svg.png[Stylingbar point cross.svg,width=16,height=16] krzyżyk,
image:16px-Stylingbar_point_down.svg.png[Stylingbar point down.svg,width=16,height=16] strzałka,
image:16px-Stylingbar_point_diamond_empty.svg.png[Stylingbar point diamond empty.svg,width=16,height=16] romb) oraz jego rozmiaru.
* image:16px-Stylingbar_line_solid.svg.png[Stylingbar line solid.svg,width=16,height=16] *Styl linii*: Wybór stylu linii (np.
image:16px-Stylingbar_line_dashed_long.svg.png[Stylingbar line dashed
long.svg,width=16,height=16] kreskowana, image:16px-Stylingbar_line_dotted.svg.png[Stylingbar line
dotted.svg,width=16,height=16] kropkowana) oraz jej grubości.
* image:16px-Stylingbar_color_white.svg.png[Stylingbar color white.svg,width=16,height=16] *Kolor obiektu*:Zmiana koloru zaznaczonego obiektu.
* image:16px-Stylingbar_color_brown_transparent_20.svg.png[Stylingbar color brown transparent 20.svg,width=16,height=16]
*Kolor i przezroczystość wypełnienia*: Zmiana koloru i przezroczystości wypełnienia obiektu.
* image:16px-Stylingbar_text.svg.png[Stylingbar text.svg,width=16,height=16] *Styl tekstu*: Ustawienie
image:16px-Stylingbar_text_color.svg.png[Stylingbar text color.svg,width=16,height=16] _koloru tekstu_,
image:16px-Stylingbar_color_white.svg.png[Stylingbar color white.svg,width=16,height=16] _koloru tła_, stylu:
(image:16px-Stylingbar_text_bold.svg.png[Stylingbar text bold.svg,width=16,height=16] *pogrubienie*,
image:16px-Stylingbar_text_italic.svg.png[Stylingbar text italic.svg,width=16,height=16] _kursywa_) oraz
image:16px-Menu-options-font-size.svg.png[Menu-options-font-size.svg,width=16,height=16] rozmiaru czcionki obiektów tekstowych.
* image:16px-Menu-options-labeling.svg.png[Menu-options-labeling.svg,width=16,height=16] *Etykieta*: Możesz wybrać następujące xref:/Etykiety_i_Opisy.adoc[ustawienia etykiety]
** _Ukryty_: Etykieta nie jest wyświetlana.
** _Nazwa_: Wyświetlana jest tylko nazwa obiektu (np. _A_).
** _Nazwa i wartość_: Wyświetlane są nazwa i wartość obiektu (np. _A = (1, 1)_).
** _Wartość_: Wyświetlana jest tylko wartość obiektu (np. _(1, 1)_).
* image:16px-Stylingbar_caption.svg.png[Stylingbar caption.svg,width=16,height=16] *Opis*: Wyświetlenie
xref:/Etykiety_i_Opisy.adoc[opisu] różnego od nazwy lub wartości obiektu (np. gdy chcesz by kilka
obiektów miało taką samą etykietę). Opis jest wprowadzany w
image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] _xref:/Okno_Ustawień_Obiektu.adoc[oknie Ustawienia obiektu]_.
* image:Pin.png[Pin.png,width=16,height=16] *Położenie bezwzględne na ekranie*: Przypięcie obiektu (np. pola tekstowego) do ekranu,
niezależnie od xref:/tools/Przemieszczaj_obszar_roboczy.adoc[_przesuwania_] lub _skalowania_ Widoku Grafiki lub Widoku Grafiki 3D (GeoGebra Desktop).
* image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] *Otwórz okno Ustawienia*: Otwiera
image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16] _xref:/Okno_Ustawień_Obiektu.adoc[okno Ustawienia]_
(GeoGebra Web i aplikacje mobilne).
* image:16px-Stylingbar_dots.svg.png[Stylingbar dots.svg,width=16,height=16] *Pokaż widoki*: Możesz wyświetlić dodatkowe
xref:/Podstawowe_widoki.adoc[_widoki_] w oknie GeoGebry (GeoGebra Web i aplikacje mobilne).
