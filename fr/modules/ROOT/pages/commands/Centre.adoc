Centre( <Conique> )::
  Retourne le centre d'un cercle,d'une ellipse ou d'une hyperbole.

[EXAMPLE]
====

*Exemple :* `++Centre(x² + 4 y² + 2x - 8y + 1 = 0)++` retourne le point _(-1, 1)_

====

littéral : `++Centre( x² + y² - 2  a x - 2 b y  +a²  + b² -1 = 0) ++` retourne (a,b)
