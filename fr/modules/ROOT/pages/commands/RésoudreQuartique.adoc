RésoudreQuartique( <Polynôme de degré 4> )::
  Retourne la liste des racines réelles ou complexes du polynôme du quatrième degré donné.

[EXAMPLE]
====

*Exemple :*

`++RésoudreQuartique[ x^4 + x^3 + x^2 + x ]++` retourne \{0, -1, _i_, -_i_ }
