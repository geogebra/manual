= Commande EnPoint
:page-en: commands/ToPoint

EnPoint( <Nombre complexe z > )::
  Crée le point (PartieRéelle = Re(z), PartieImaginaire = Im(z)) représentatif du nombre complexe z.

[EXAMPLE]
====

`++EnPoint(3+2ί)++` retourne le point (3,2).

====
