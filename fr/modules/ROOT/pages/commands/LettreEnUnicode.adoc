LettreEnUnicode( <"Lettre"> )::
  Transforme une simple lettre en son https://fr.wikipedia.org/Unicode[code Unicode].

[NOTE]
====

La lettre doit être entre guillemets.

====

[EXAMPLE]
====

`++LettreEnUnicode("a")++` retourne le nombre 97.

====
