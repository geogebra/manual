Parabole( <Point F>, <Droite d> )::
  Parabole de foyer _F_ et de directrice _d_.

[EXAMPLE]
====

*Exemple :* Soit `++a = Droite((0,1), (2,1))++`. `++Parabole((3, 3), a)++` retourne la parabole

dont, en fonction de votre choix, l'équation affichée sera :

_x² - 6x - 4y = -17_

_x² = 6x + 4y - 17_

_y = 0.25x² - 1.5x +4.25_

_y = 0.25(x-3)² + 2_

_4(y - 2) = (x - 3)²_.

====
