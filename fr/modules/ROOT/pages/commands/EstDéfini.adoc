= Commande EstDéfini
:page-en: commands/IsDefined
ifdef::env-github[:imagesdir: /fr/modules/ROOT/assets/images]

EstDéfini(<Objet>)

Retourne _true_ (vrai) ou _false_ (faux) suivant que l’objet est défini ou non.

[EXAMPLE]
====

`++EstDéfini(Cercle((1,1), -2))++` retourne _false_.

====
