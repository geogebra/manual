EnBas( <Quadrique Limitée > )::
  Crée le "fond" pour une quadrique limitée (par ex. un cercle pour un cylindre).

[EXAMPLE]
====

`++EnBas(Cylindre(A, B, 2))++` crée le cercle d'axe (AB), de centre A et de rayon 2.
