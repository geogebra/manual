====

Activez l'outil image:32px-Mode_slider.svg.png[Mode slider.svg,width=32,height=32], puis cliquez ou tapez sur une place vide dans la vue image:16px-Menu_view_graphics.svg.png[Menu view
graphics.svg,width=16,height=16]xref:/Graphique.adoc[Graphique] pour y créer un curseur pour un nombre ou un angle.


Une fenêtre de dialogue s’ouvre pour y préciser :

* s’il s’agit d’un _Nombre_, d’un _Angle_ ou d’un _Entier_, s’il est _Aléatoire_, son _Intervalle_ : [_min_ ; _max_] et son _Incrément_ ; 

* Dans l’onglet « Curseur » : s’il est _fixé_, ainsi que son orientation (_horizontal_ / _vertical_) 
** si un curseur est _fixé_ tout clic dessus fera varier sa valeur.


* et Dans l’onglet «  d'Animation » sa _Vitesse_ et son mode d'animation  _Répéter_ : stem:[\Leftrightarrow] _Alterné_ / stem:[\Rightarrow] _Croissant_ / stem:[\Leftarrow] _Décroissant_ / stem:[\Rightarrow] _Une fois (croissant)_
 
====

Les paramètres _min_, _max_, _Incrément_ et _Vitesse_ sont dynamiques, vous pouvez utiliser des [.mw-selflink .selflink]#curseurs#.

====

[NOTE]
====

Dans la fenêtre de dialogue Curseur, vous pouvez entrer le symbole degré ° ou _pi (π)_ pour l’intervalle et
l’incrément en utilisant les raccourcis claviers suivants :

* _Alt-O_ (MacOS: _Ctrl-O_) pour le symbole degré °
* _Alt-P_ (MacOS: _Ctrl-P_) pour le symbole _pi (π)_

====

La position du curseur peut être absolue dans la vue image:16px-Menu_view_graphics.svg.png[Menu view
graphics.svg,width=16,height=16]xref:/Graphique.adoc[Graphique] (cela signifie que le curseur n’est pas affecté par le

graphics.svg,width=16,height=16]xref:/Graphique.adoc[Graphique]) ou relative au repère de coordonnées (Ce paramètre peut être modifié ultérieurement dans la boîte de dialogue
image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16]  xref:/Dialogue_Propriétés.adoc[Propriétés] du nombre ou angle correspondant).

[NOTE]

Si un curseur est animé, un bouton apparaît dans le coin inférieur gauche de la vue Graphique, vous permettant
soit de mettre l’animation en image:Animate_Pause.png[Animate Pause.png,width=16,height=16] pause, soit de poursuivre
image:Animate_Play.png[Animate Play.png,width=16,height=16] l’animation.

*Sauf si*, dans les propriétés, vous avez défini _Vitesse_ par une variable, dans ce cas le bouton n'est pas affiché.

xref:/Accessibilité.adoc[Accessibilité] : La touche [.kcode]#espace# permet elle aussi de suspendre/reprendre
l'animation d'un curseur.

== Curseurs fixés

Les curseurs créés avec l'Outil image:32px-Mode_slider.svg.png[Mode slider.svg,width=32,height=32] *Curseur*
ne sont pas fixés par défaut. Comme tous les autres objets, ils peuvent être xref:/Propriétés_d_un_objet.adoc[fixés].

Sélectionnez le menu contextuel ou le dialogue image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16]  xref:/Dialogue_Propriétés.adoc[Propriétés] pour modifier ce paramètre et fixer le curseur.

Une fois fixé, le curseur reste déplaçable. 


Pour déplacer un curseur fixé :

Sur un écran tactile :  Activez l'outil image:32px-Mode_slider.svg.png[Mode slider.svg,width=32,height=32] Curseur et faites glisser le curseur ;

Sur un écran non tactile :

* quand l'outil image:32px-Mode_move.svg.png[Mode move.svg,width=32,height=32] xref:/tools/Déplacer.adoc[Déplacer] 
est sélectionné, vous pouvez le glisser à l'aide du bouton droit de la souris ; 
* quand l'outil image:32px-Mode_slider.svg.png[Mode slider.svg,width=32,height=32] *Curseur* est sélectionné, 
vous pouvez utiliser aussi bien le bouton droit que le gauche. 

== Propriétés d’un curseur :
 
En plus des propriétés déjà citées ci-dessus, dans image:16px-Menu-options.svg.png[Menu-options.svg,width=16,height=16]  xref:/Dialogue_Propriétés.adoc[Propriétés], vous pouvez aussi définir

* _Style Point_ : _Taille_ et _Couleur_ ;
* _Style du trait_ : _Largeur_, _Épaisseur du trait_, _Couleur_ et _Opacité Tracé_
