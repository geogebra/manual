== CAS での書式

SolveQuartic( <Quartic Polynomial> )::
  与えられた https://ja.wikipedia.org/%E5%9B%9B%E6%AC%A1%E5%87%BD%E6%95%B0[四次多項式]
  を解き，すべての解のリストを返す．

[EXAMPLE]
====

`++SolveQuartic[ x^4 + x^3 + x^2 + x ]++` 出力： \{0, -1, _i_, -_i_ }

====
