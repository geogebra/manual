== CAS での書式

MixedNumber( <数値> )::
  指定された数を帯分数（混合数）に変換する．

[EXAMPLE]
====

* `++MixedNumber(3.5)++` 出力： stem:[3 + \frac\{1}\{2}].
* `++MixedNumber(12 / 3)++` 出力： _4_.
* `++MixedNumber(12 / 14)++` 出力： stem:[\frac\{6}\{7}].

====

====
